<Steps>
<Step title="Création d’une ville">
Pour créer une ville, il suffit de faire la commande suivante :
```
/ville create <nom>
```
<Tip> Indiqué le nom que vous voulez à la place de \<nom></Tip>
</Step>


<Step title="Faire son premier claim">

Une fois votre ville construite, vous pouvez claim une zone dans le monde [construction](/monde/construction).
Par défaut, vous ne pouvez pas interagir avec le monde construction en dehors de vos claims ou des claims ou vous êtes membres.
<Note>Un claim fait exactement un chunk de minecraft, à savoir un carré de 16 par 16.
Vous pouvez les voirs à partir de la commande F3 + G.</Note>
</Step>

<Step title="Information supplémentaire">
- /ville -> **/v**
- Nombre maximum de claim par ville : **150**
- Invité un joueur -> **/ville invite \<pseudonyme>**
- Exclure un joueur -> **/ville kick \<pseudonyme>**
- Voir les bordures de son claim -> **/v view** *(Nécessite d'activer les particules dans les options)*
</Step>
